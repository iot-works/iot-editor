#IoT Editor(物联网编辑器)

> A Internet of Things Editor

![Screenshot](screenshot.jpg)

## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
